Detailed instructions on how to obtain a Hypixel API key

If you are already registered on the website, you can skip Chapters 1 and 2. I also strongly recommend that everyone goes through Chapter 4.

Estimated time for this step is 1-48 hours

Those who already have an account on the website can skip this chapter.

Obsolete supporting literature: https://support.hypixel.net/hc/en-us/articles/360019646639

Step 1: Go to the website: https://hypixel.net/register/
Step 2: Fill in the required forms.
Step 3: Press the registration button.
Step 4: Wait less than 48 hours for your account to be verified.

Estimated time for this step is 10 minutes

For those who have already linked Minecraft to Hypixel Forum, skip this step.

Step 1: Click on the link: https://hypixel.net/account/minecraft
Step 2: Everything is described on the official website, read the following link: https://support.hypixel.net/hc/en-us/articles/360019647059
Step 3: The result to be achieved

Estimated time for this step is 20 minutes

This step is necessary so that you can play until your 'Personal API Key' request is approved. This key is given for 3 day, so I highly recommend applying for a 'Personal API Key'.

Step 1: Go to: https://developer.hypixel.net/
Step 2: Press the "Login" button
Step 3: Click the 'Create API key' button. You may already have a generated key, then skip this step.
Step 4: Copy the key. Everyone copies it differently, I suggest clicking on the green rectangle to put the key on the clipboard. If you can't copy it, enter the key manually directly into Minecraft.
Step 5: Enter the command «@api <Your development key>» into the chat. Instead of «<Your development key>» enter your key. Example: «@api c14a1ecd-686d-45d8-8c49-1a8453d090d9»
Step 6: Execute the command, if the command fails then you have not rewritten the key correctly.

Estimated time for this step is 1-72 hours

This step is necessary if you want a permanent key.

Step 1. Go to https://developer.hypixel.net/dashboard/ and click on "Create App".
Step 2: Next, click "Continue" under "Personal API Key".
Step 3: Hypixel's policy changes all the time, so read it before writing a description. I can't guarantee you'll get the key, I'm just suggesting you lie a little. At this point you can introduce yourself as the developer of your own mod. Be sure to come up with a name and description yourself, it will be fun if there are many developers of the same mod.
Step 4: Press "Continue" and any other confirmation buttons that appear.
Step 5: Wait 1-48 hours for the application to be confirmed. Wait for the icons I have highlighted in red to disappear.
Step 6: After waiting for the request to be approved, click on the "Regenerate API Key" button.
Step 7: Click "Confirm"
Step 8: Copy the key. Everyone copies it differently, I suggest clicking on the green rectangle to put the key on the clipboard. If you can't copy it, enter the key manually directly into Minecraft.
Step 9: Enter the command «@api <Your development key>» into the chat. Instead of «<Your development key>» enter your key. Example: «@api c14a1ecd-686d-45d8-8c49-1a8453d090d9»
Step 10: Execute the command, if the command fails then you have not rewritten the key correctly.
